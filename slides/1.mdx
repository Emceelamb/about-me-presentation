# Peace and Security Data Hub
### United Nations - DPPA-DPO-IMU


---

<div className="mx-20">

# Context

Peace and Security data is collected by various departments, teams, and missions throughout the United Nations and saved internally. The Information Management Unit (IMU) acts as a data router for the pillar to respond to data requests from various departments. The manual pipeline creates a lengthy response time, redundancies in data collection, and data inaccuracies.

</div>

---

<div className="mx-20">

# Project

The IMU set forth to create a Peace and Security Data Hub, an infrastructure to standardize data, central database, and APIs to serve the data. The Data Hub is a publicly available website that allows both UN staff and researchers to browse data sets, preview metadata, preview data, download, and use the data via APIs. Additionally, data providers can add editorial content to contextualize the data. 

</div>
